Quality assessment

All data have been compared with the current Harmonic Constants library, and the residuals checked. Records have been visually inspected for spikes, datum shifts and timing errors.
Where there are problems with the primary channel, data may be available from the backup channel. Please contact enquiries@bodc.ac.uk for more information.
In March 1986 the DATARING system was installed. There is a gap in the file where the gauge was changed.
In December 1996, the channel specifications were changed and the primary channel was altered from channel two (ASLVZZ02) to channel one (ASLVZZ01). There is a gap in the file where the channels were switched.
In May 1997, the backup bubbler gauge (ASLVBG02) became the primary channel, and what had been the primary gauge (ASLVZZ01) became the backup channel. There is a gap in the file where the gauge was changed.
February 2003 - File output has been altered so that all values fall on the quarter hour and gaps are filled in with null values, marked with an 'N' flag.
March 2003 - Residuals (observed minus predicted sea level values, measured in metres) now included in the file output.
On 26/01/2012 a new data logger was installed. Tests were carried out to ensure the logger was functioning correctly, resulting in some data being lost.
November 2012 - The file output has been altered to reflect the Contributor name change from Proudman Oceanographic Laboratory to National Oceanography Centre, Liverpool. Also, the Parameter code has been improved and now uses a controlled vocabulary.
